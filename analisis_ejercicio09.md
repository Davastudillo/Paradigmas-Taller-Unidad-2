1. Analisis ejercicio 9:

Resultado: {1, 3, 4}

¿Qué retorna el bloque de código? 
El bloque de código retorna el conjunto resultante de las operaciones, en este caso {1, 3, 4}

¿Cuántas funciones se declararon? 
Se declararon 2 funciones: agregar_elemento y eliminar_elemento

¿Qué imprime en consola? 
No imprime en consola, solo retorna el conjunto resultante de las operaciones.

2. Analisis del codigo:

Inicialmente se declaran las funciones agregar_elemento y eliminar_elemento con su respectivo parametro.

Luego se declara la variable conjunto con su respectivo valor

Luego se realiza el calculo, la función agregar_elemento agrega el elemento 4 al conjunto, y la función eliminar_elemento elimina el elemento 2 del conjunto.

Finalmente se retorna el conjunto resultante de las operaciones.
